DESCRIPTION:

This project focuses on solving a real-world business problem using Linear Programming to optimize product manufacturing decisions. The goal is to determine how many units of each product should be produced in order to maximize total profit while staying within limited resources such as machine hours, labor hours, and raw materials. A dataset containing multiple products with their respective profit values and resource requirements is used to build a mathematical model. The problem is implemented in Python using the PuLP library, which efficiently finds the optimal solution based on the defined objective function and constraints.

The results of the model provide an optimal production plan that ensures maximum profitability with efficient utilization of available resources. The project also highlights important practical insights, such as how optimization models may favor certain products more than others and why adding realistic constraints is necessary for balanced production. Overall, this project demonstrates how optimization techniques can support data-driven decision-making in business environments and can be extended further for more complex scenarios like supply chain or multi-factory planning.
